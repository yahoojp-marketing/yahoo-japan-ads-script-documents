### template

• **template**: [*Sheet*](sheet.md)

シートをコピーする際に指定するコピー元シートです。<br>シートオブジェクトのすべてのデータが新しいシートにコピーされます。<br>指定するシートは、スプレッドシートオブジェクトのシートの1つである必要があります。
